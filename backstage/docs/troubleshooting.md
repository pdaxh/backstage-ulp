# Troubleshooting Guide

This guide helps you resolve common issues with the ULP Backstage Developer Portal.

## Common Issues

### 401 Unauthorized Error

**Problem**: You're seeing a 401 Unauthorized error when accessing documentation.

**Solution**: 
- Ensure you're logged in to Backstage
- Check if you have the necessary permissions
- Try refreshing the page
- Clear your browser cache

### Search Not Working

**Problem**: Search functionality is not returning results.

**Solution**:
- Check your search terms
- Try broader or more specific keywords
- Ensure the service you're looking for is properly indexed
- Contact your platform team if the issue persists

### Service Not Appearing in Catalog

**Problem**: A service you expect to see is not showing up in the catalog.

**Solution**:
- Verify the service has a proper `catalog-info.yaml` file
- Check if the service is properly registered
- Ensure the service metadata is correct
- Contact the service owner

### Documentation Not Loading

**Problem**: Documentation pages are not loading or showing errors.

**Solution**:
- Check if the documentation source is accessible
- Verify the documentation format is correct
- Ensure proper permissions are set
- Try accessing the documentation directly

## Browser Issues

### Page Not Loading

**Solution**:
- Clear browser cache and cookies
- Try a different browser
- Check your internet connection
- Disable browser extensions temporarily

### Styling Issues

**Solution**:
- Hard refresh the page (Ctrl+F5 or Cmd+Shift+R)
- Clear browser cache
- Check if JavaScript is enabled
- Try incognito/private browsing mode

## Performance Issues

### Slow Loading

**Solution**:
- Check your internet connection
- Try during off-peak hours
- Clear browser cache
- Contact your platform team if the issue persists

### Search Timeouts

**Solution**:
- Try more specific search terms
- Reduce the scope of your search
- Wait a moment and try again
- Contact support if the issue continues

## Getting Help

### Self-Service Options

1. **Check this guide** for common solutions
2. **Search existing documentation** for similar issues
3. **Check service status** if available
4. **Review recent changes** that might affect functionality

### Contact Support

If you can't resolve the issue:

1. **Document the problem**:
   - What were you trying to do?
   - What error messages did you see?
   - What browser are you using?
   - When did the issue occur?

2. **Contact your platform team**:
   - Use the appropriate communication channels
   - Provide the information you documented
   - Include screenshots if helpful

3. **Check for updates**:
   - Look for announcements about maintenance
   - Check if there are known issues
   - Review recent changes or updates

## Best Practices

### Preventing Issues

- **Keep your browser updated**
- **Clear cache regularly**
- **Use supported browsers**
- **Follow documentation guidelines**

### Reporting Issues

- **Be specific** about what you were doing
- **Include error messages** and screenshots
- **Mention your browser** and operating system
- **Describe steps to reproduce** the issue

---

*If you continue to experience issues, please contact your platform team with the details above.*
