# Getting Started with ULP Backstage

Welcome to the ULP Backstage Developer Portal! This guide will help you get up and running quickly.

## What is ULP Backstage?

ULP Backstage is a developer portal that provides a unified interface for:

- **Service Discovery**: Find and explore all your services and APIs
- **Documentation**: Access technical documentation and guides
- **Software Templates**: Create new projects using standardized templates
- **API Management**: Discover and test APIs across your organization

## First Steps

### 1. Explore the Catalog

Start by browsing the **Catalog** to see what services and components are available in your organization.

### 2. Search for Resources

Use the **Search** functionality to quickly find specific services, documentation, or people.

### 3. Create New Components

Use the **Create** button to scaffold new services using our predefined templates.

### 4. Access Documentation

Visit the **Docs** section to access technical documentation and guides.

## Navigation

- **Home**: Your personalized dashboard
- **Catalog**: Browse all services and components
- **Docs**: Technical documentation
- **Create**: Build new components
- **Search**: Find anything across the organization
- **Settings**: Configure your preferences
